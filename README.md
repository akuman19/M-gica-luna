# Luna Andina Glamping

Bienvenido al repositorio de **Luna Andina Glamping**, una experiencia de lujo en armonía con la naturaleza en el Valle del Cauca, Colombia.

## Descripción

Este proyecto es el sitio web oficial de Luna Andina, desarrollado con React, TypeScript y Tailwind CSS.

## Características

- Diseño moderno y responsivo.
- Galería de imágenes.
- Información detallada de alojamientos (Tienda Safari, Domo Geodésico, Cabaña Luna).
- Integración con Google Maps.
- Reservas vía WhatsApp.

## Instalación

1.  Clonar el repositorio.
2.  Ejecutar `npm install`.
3.  Ejecutar `npm run dev` para iniciar el servidor de desarrollo.

## Tecnologías

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
